## Number row characters

- ~ : Tilde
- ! : Exclamation Mark
- @ : At Sign
- \# : Number Sign or Pound
- $ : Dollar Sign
- % : Percent Sign
- ^ : Caret
- & : Ampersand
- \* : Asterisk
- ( : Left Parenthesis
- ) : Right Parenthesis
- \_ : Underscore
- \+ : Plus Sign
